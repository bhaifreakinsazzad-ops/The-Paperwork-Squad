# The Paperwork Squad

This repository contains a Vite + React + TypeScript project for The Paperwork Squad.

## Current state
- Public landing page
- Auth modal
- Consultation modal
- Fundability checker modal
- Valuation modal
- Client portal
- Admin dashboard

## Run locally
1. Install dependencies
2. Start development server with `npm run dev`
3. Build with `npm run build`

## Notes
- Add your own environment values before production use.
- `env.example.txt` contains placeholder environment variables.
